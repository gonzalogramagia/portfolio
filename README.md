# 🚀 Gonzalo Gramagia's Portfolio

**A modern, high-performance personal portfolio website.**

This project serves as my digital identity, showcasing my work experience, projects, and blog posts. Built with the latest web technologies, it emphasizes performance, accessibility, and a premium user experience.

---

## ✨ Key Features

- **🌍 Internationalization (i18n)**: Seamless English and Spanish content support.
- **🌗 Dark Mode**: Integrated theme switching with persistent state preference.
- **🏃‍♂️ Animated UI**: Smooth transitions and micro-interactions powered by Motion.
- **📱 Responsive Design**: Fully optimized for all device sizes, from mobile to desktop.
- **📝 Blog Integration**: Markdown-based blog system with reading time and categorization.
- **🧭 Enhanced Navigation**: Custom table of contents and smooth navigation for blog posts.
- **🔍 Interactive Media**: Expandable images and video previews for detailed project viewing.
- **🔔 Toast Notifications**: Contextual alerts for user interactions (e.g., copying email).
- **⚡ High Performance**: Built on Next.js 16 with React Server Components.

---

## 🛠️ Technology Stack

- **Framework**: [Next.js 16](https://nextjs.org/) + [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Animations**: [Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Notifications**: [Sonner](https://sonner.emilkowal.ski/)
